# Changelog

All notable template changes should be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project follows [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Initial standalone template scaffold based on a production academic homepage.
- Bilingual content examples under `content/`.
- Placeholder avatar and CV assets under `public/`.
- Documentation for content editing, maintenance, Vercel deployment, and optional automation.

### Changed

- Removed personal profile data, links, and deployment defaults.
- Replaced hard-coded identity strings in metadata, contact page, and footer.
- Simplified the automation workflow to make it optional and template-safe.
